## Design Context

### Users
Primary user is the creator (personal use). The context is a personal productivity workspace for managing chat, image galleries, documents, and files. The job to be done is centralized personal organization and asset management.

### Brand Personality
Calm, clinical, precise. The interface should evoke a sense of quiet focus and control, acting as a clean canvas rather than a loud distraction.

### Aesthetic Direction
A refined, minimal aesthetic that supports both light and dark modes. It should feel highly functional, structured, and deliberate without unnecessary ornamentation.

### Design Principles
1. **Restraint over decoration**: Every visual element must serve a functional purpose; remove noise to highlight the content (chats, images, docs).
2. **High-precision typography**: Use a refined typographic scale that feels clinical and exact, prioritizing readability and dense information display when needed.
3. **Calm color palette**: Use desaturated neutrals tinted towards a subtle, calming hue (e.g., slate or cool gray) with very restrained accent colors.
4. **Fluid adaptability**: The layout should gracefully adapt to the active module (chat, gallery, docs) while maintaining a consistent mental model across the workspace.
